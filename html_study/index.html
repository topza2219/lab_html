เรียนรู้เกี่ยวกับแท็ก <head> ใน HTML

ในบทเรียนนี้ เราจะเรียนรู้เกี่ยวกับการใช้แท็ก <head> เพื่อเก็บข้อมูลเมตา ข้อมูลสไตล์ชีท และสคริปต์ต่างๆ ที่ไม่แสดงผลบนหน้าเว็บโดยตรง

แท็ก <head> เป็นส่วนสำคัญของเหตุการณ์ HTML ซึ่งช่วยให้เบราว์เซอร์และเครื่องมือค้นหาเข้าใจเนื้อหาของหน้าเว็บได้ดีขึ้น

ใช่สำหรับกรอกค่าและเก็บทางแป้นพิมพ์

ตัวอย่างเช่น <input> <textarea> <select> เป็นต้น

กราฟิกใน HTML

เรียนรู้เกี่ยวกับการใช้กราฟิก เช่น <canvas> <svg> เป็นต้น

วิดิโอใน HTML

เรียนรู้เกี่ยวกับการฝังวิดิโอใน HTML โดยใช้แท็ก <video>

API ใน HTML

เรียนรู้เกี่ยวกับการใช้ API ต่างๆ ใน HTML เช่น Geolocation API, Web Storage API เป็นต้น

รูปภาพใน HTML

เรียนรู้เกี่ยวกับการใช้รูปภาพใน HTML โดยใช้แท็ก <img>

แท็ก title ใน HTML

เรียนรู้เกี่ยวกับการใช้แท็ก <title> เพื่อกำหนดชื่อของหน้าเว็บ

ตารางใน HTML

เรียนรู้เกี่ยวกับการสร้างตารางใน HTML โดยใช้แท็ก <table>

รายการใน HTML

เรียนรู้เกี่ยวกับการสร้างรายการใน HTML โดยใช้แท็ก <ul>, <ol>, <li>

บล็อกใน HTML

เรียนรู้เกี่ยวกับการใช้แท็กบล็อก เช่น <div>, <section>, <article> เป็นต้น

แท็ก div ใน HTML

เรียนรู้เกี่ยวกับการใช้แท็ก <div> เพื่อจัดกลุ่มเนื้อหาในหน้าเว็บ

แท็ก id ใน HTML

เรียนรู้เกี่ยวกับการใช้แท็ก <id> เพื่อกำหนดตัวระบุที่ไม่ซ้ำกันให้กับองค์ประกอบในหน้าเว็บ

JavaScript ใน HTML

เรียนรู้เกี่ยวกับการใช้ JavaScript เพื่อเพิ่มฟังก์ชันการทำงานให้กับหน้าเว็บ

Emoji ใน HTML

เรียนรู้เกี่ยวกับการใช้ Emoji ใน HTML เพื่อเพิ่มความน่าสนใจให้กับเนื้อหา

สัญลักษณ์ใน HTML

เรียนรู้เกี่ยวกับการใช้สัญลักษณ์ใน HTML เพื่อแสดงตัวอักษรพิเศษที่ไม่สามารถพิมพ์ได้โดยตรง
