เรียนรู้เกี่ยวกับรูปภาพใน HTML

ในบทเรียนนี้ เราจะเรียนรู้เกี่ยวกับการใช้รูปภาพใน HTML โดยใช้แท็ก <img> เพื่อแสดงรูปภาพบนเว็บเพจ

แท็ก <img> ใช้สำหรับฝังรูปภาพในหน้าเว็บ ซึ่งสามารถกำหนดแหล่งที่มาของรูปภาพด้วยแอตทริบิวต์ src และกำหนดข้อความสำรองด้วยแอตทริบิวต์ alt

ตัวอย่างโค้ด HTML สำหรับแท็ก <img>

        <img src="path/to/image.jpg" alt="คำอธิบายรูปภาพ" width="600" height="400">

ในตัวอย่างข้างต้น เราได้กำหนดแหล่งที่มาของรูปภาพเป็น "path/to/image.jpg" และเพิ่มแอตทริบิวต์ alt เพื่อให้คำอธิบายรูปภาพในกรณีที่รูปภาพไม่สามารถโหลดได้ นอกจากนี้ เรายังสามารถกำหนดความกว้างและความสูงของรูปภาพได้ด้วยแอตทริบิวต์ width และ height
