เรียนรู้เกี่ยวกับกราฟิก

ในบทเรียนนี้ เราจะเรียนรู้เกี่ยวกับการใช้กราฟิกใน HTML เช่น <canvas> และ <svg> เพื่อสร้างภาพและกราฟิกต่างๆ บนเว็บเพจ

องค์ประกอบ <canvas> ของ HTML ใช้สำหรับ วาดกราฟิกแบบเรียลไทม์ โดยใช้ JavaScript

องค์ประกอบ <svg> ของ HTML ใช้สำหรับ สร้างกราฟิกแบบเวกเตอร์ที่สามารถปรับขนาดได้โดยไม่สูญเสียความคมชัด

Canvas มี เมธอดหลายแบบ สำหรับการวาด เส้นทาง, กล่อง, วงกลม, ข้อความ และการ เพิ่มรูปภาพ

SVG ใช้แท็กต่างๆ เช่น <circle>, <rect>, <line>, <text> เพื่อสร้างกราฟิกที่ซับซ้อน

ตัวอย่างโค้ด JavaScript สำหรับการวาดกราฟิกบน <canvas>:

        <!-- สร้างพื้นที่วาดกราฟิกขนาด 200x100 พิกเซล พร้อมกรอบสีดำ -->
        <canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>

        <script>
        // ดึง element canvas
        var c = document.getElementById("myCanvas");
        // ตั้งค่า context สำหรับวาดแบบ 2D
        var ctx = c.getContext("2d");
        // กำหนดสีที่จะเติมเป็นสีแดง
        ctx.fillStyle = "#FF0000";
        // วาดสี่เหลี่ยมเริ่มจากพิกัด (20,20) ขนาด 150x55 พิกเซล
        ctx.fillRect(20, 20, 150, 55);
        </script>
