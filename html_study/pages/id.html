เรียนรู้เกี่ยวกับแท็ก <id> ใน HTML

ในบทเรียนนี้ เราจะเรียนรู้เกี่ยวกับการใช้แท็ก <id> เพื่อกำหนดตัวระบุที่ไม่ซ้ำกันให้กับองค์ประกอบในหน้าเว็บ

แท็ก <id> ใช้สำหรับกำหนดตัวระบุที่ไม่ซ้ำกันให้กับองค์ประกอบในหน้าเว็บ ซึ่งช่วยให้เราสามารถเข้าถึงและจัดการกับองค์ประกอบนั้นได้ง่ายขึ้นผ่านทาง CSS หรือ JavaScript

ตัวอย่างโค้ด HTML สำหรับแท็ก <id>

        <div id="unique-id">
            <h1>หัวข้อหลัก</h1>
            <p>นี่คือย่อหน้าที่มี id</p>
        </div>

ในตัวอย่างข้างต้น เราได้สร้างบล็อกที่มี id ชื่อ "unique-id" ซึ่งประกอบด้วยหัวข้อหลักและย่อหน้าหนึ่งย่อหน้า

ตัวอย่างบล็อกที่แสดงผลลัพธ์:

หัวข้อหลัก

นี่คือย่อหน้าที่มี id
