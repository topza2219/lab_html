เรียนรู้เกี่ยวกับ JavaScript ใน HTML

ในบทเรียนนี้ เราจะเรียนรู้เกี่ยวกับการใช้ JavaScript เพื่อเพิ่มฟังก์ชันการทำงานให้กับหน้าเว็บ

JavaScript เป็นภาษาการเขียนโปรแกรมที่ใช้ในการพัฒนาเว็บ ซึ่งช่วยให้เราสามารถสร้างหน้าเว็บที่มีความโต้ตอบและมีฟังก์ชันการทำงานที่ซับซ้อนมากขึ้น

ตัวอย่างโค้ด HTML ที่ใช้ JavaScript

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>JavaScript Example</title>
            <script>
                function showMessage() {
                    alert("Hello, this is a JavaScript alert!");
                }
            </script>
        </head>
        <body>
            <button onclick="showMessage()">Click Me</button>
        </body>
        </html>

ในตัวอย่างข้างต้น เราได้สร้างปุ่มที่เมื่อคลิกจะเรียกใช้ฟังก์ชัน JavaScript ที่แสดงกล่องแจ้งเตือน
