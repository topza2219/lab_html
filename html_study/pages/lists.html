เรียนรู้เกี่ยวกับรายการใน HTML

ในบทเรียนนี้ เราจะเรียนรู้เกี่ยวกับการสร้างรายการใน HTML โดยใช้แท็ก <ul> สำหรับรายการที่ไม่มีลำดับ และแท็ก <ol> สำหรับรายการที่มีลำดับ

แท็ก <ul> ใช้สำหรับสร้างรายการที่ไม่มีลำดับ ซึ่งแต่ละรายการจะถูกแสดงด้วยเครื่องหมายจุด (bullet points)

ตัวอย่างโค้ด HTML สำหรับแท็ก <ul>

        <ul>
            <li>รายการที่ 1</li>
            <li>รายการที่ 2</li>
            <li>รายการที่ 3</li>
        </ul>

ในตัวอย่างข้างต้น เราได้สร้างรายการที่ไม่มีลำดับที่มีสามรายการ

แท็ก <ol> ใช้สำหรับสร้างรายการที่มีลำดับ ซึ่งแต่ละรายการจะถูกแสดงด้วยตัวเลขหรือตัวอักษรที่เรียงลำดับ

        <ol>
            <li>รายการที่ 1</li>
            <li>รายการที่ 2</li>
            <li>รายการที่ 3</li>
        </ol>
